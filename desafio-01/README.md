# Desafio Ignite ToDo

Desafio proposto na trilha do Ignite React para aplicar os conhecimentos adquiridos.

## Tecnologias utilizadas:

* React
* JS
* HTML / CSS
* Phospher Icons
* LocalStorage

## Como rodar o projeto?

    * "git clone"
    * "npm i"
    * no terminal rodar: "npm run dev"
    * abrir a url no navegador!
